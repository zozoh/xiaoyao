函数练习 0701 - 改变字符串

在一个名为 func_0701.js 里提供提供一个函数:
    function func_0701(str) {
        ...
        return 改变后的字符串 
    }
要求：
 1. 给你的参数 str 表示一个字符串
 2. 你需要对其进行改动，并返回改动后的字符串，改动的要求为：
    将字符串的每个字符都变成编码数字，并且
    如果一个字符编码数字大于 128，或者小于 0 ，则变成 "[??]"
比如:
    func_0701("abc");     # 应该返回 "[97][98][99]"
    func_0701("123");     # 应该返回 "[49][50][51]"
    func_0701("?我");     # 应该返回 "[63][??]"
    func_0701("明天");    # 应该返回 "[??][??]"
    func_0701(null);      # 应该返回 null
