函数练习 0628

在一个名为 func_0628.js 里提供提供一个函数:
    function func_0628(str) {
        ...
        return 反转后的字符串 
    }
要求：
 1. 给你的参数 str 表示一个字符串
 2. 你需要返回一个字符串，是给定字符串的反转串
比如:
    func_0628("abc");     # 应该返回 "cba"
    func_0628("x");       # 应该返回 "x"
    func_0628("");        # 应该返回 ""
    func_0628(null);      # 应该返回 null
